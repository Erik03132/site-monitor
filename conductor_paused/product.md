# Product Definition: Aether Monitor

## Обзор
Сервис мониторинга веб-сайтов с детектированием изменений на основе семантических чанков и суммаризацией через ИИ.

## Целевая аудитория
- Вебмастера, SEO-специалисты, владельцы интернет-магазинов.

## Проблематика
- Сложность отслеживания обновлений на множестве ресурсов вручную.
- Необходимость понимания сути изменений без ручного сравнения версий.

## Ключевые цели
1. Стабильный мониторинг сайтов с настраиваемым интервалом.
2. Точное детектирование изменений (добавлено/удалено/изменено).
3. Суммаризация изменений с помощью LLM (Perplexity/Kimi).
4. Мгновенные уведомления.
