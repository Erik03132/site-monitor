# Tech Stack: Aether Monitor

## Core
- **Framework**: Next.js 15.1 (App Router)
- **Language**: TypeScript 5.x
- **Platform**: Vercel

## UI/UX
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React / Material Symbols
- **Animations**: CSS Transitions / Tailwind utilities
- **Design System**: "Dark Luxury" Glassmorphism

## Backend & Data
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **ORM/Client**: Supabase JS SDK / @supabase/ssr
- **Content Fetching**: Fetch API + Cheerio
- **AI**: Perplexity API (Sonar Pro) / Kimi AI
