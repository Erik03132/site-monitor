# Product Definition - site-monitor

## Project Name
site-monitor

## Description
Система мониторинга веб-сайтов в реальном времени с отслеживанием изменений контента и ключевых слов.

## Problem Statement
Юзерам сложно вручную отслеживать изменения на множестве сайтов (пропуск контента, акций, ошибок).

## Target Users
Владельцы бизнеса, следящие за конкурентами.

## Key Goals
- Стабильная работа сканера без блокировок.
